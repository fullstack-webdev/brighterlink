Version 0.2.22

Release date: 01/20/2015

Upgrades and Fixes

Coming Soon

  • On second login, an Info panel, so that Users are aware of the fact that the current Dashboard is where they left off.
  • A legend label editable field to edit a widget legend label onClick of the label.
  • Help and Updates dialog in the BrighterLink App panel to view recent updates and access helpful end user information.
